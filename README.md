# Helock

Helock is personal alarm clock worker on heroku.com.  
Every mornig, helock sends direct message to you.

@helock_bot < Good Morning !

## Settings

### ENV

See [.env.sample](https://github.com/grauwoelfchen/helock/blob/master/.env.sample)

### messages.yaml

```
---
- ['Sunday :-D']
- ['Monday :-(']
- [
    'Good Morning!',
    'Wake up!',
  ]
...
```

### users.yaml

```
---
- 'grauwoelfchen'
- 'you'
```
